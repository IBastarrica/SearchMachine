# 🚀 Bash Scripting Utility: Search Machines

Este proyecto consiste en una herramienta de automatización desarrollada en **Bash**, diseñada para interactuar con un entorno de `bundle.js` y facilitar la gestión de datos mediante filtros avanzados y búsqueda de actualizaciones.

---

## 🛠️ Funcionalidades Principales

El script ofrece una interfaz por línea de comandos para ejecutar las siguientes tareas:

* **🔍 Filtrado Avanzado:** Permite segmentar la información del bundle según:
    * **Nombre:** Búsqueda específica de registros.
    * **Dirección IP:** Identificación de nodos o usuarios.
    * **Skills:** Filtro por habilidades o competencias técnicas.
    * **Sistema Operativo:** Clasificación por entorno de ejecución (Linux, Windows, MacOS).
* **🔄 Gestor de Actualizaciones:** Módulo integrado para buscar y verificar nuevas versiones del sistema.
* **📦 Integración JS:** Conexión directa y procesamiento de datos provenientes de `bundle.js`.
